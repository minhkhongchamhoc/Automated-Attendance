Introduction

Welcome to the Facial Recognition Software User Guide. This project, as detailed in the Automated Attendance README, is designed to streamline student attendance using facial recognition technology. By leveraging a robust MySQL database and employing advanced facial detection algorithms, the software enables automated, efficient, and accurate attendance tracking.

The system primarily uses the default webcam for camera connection, ensuring ease of setup while maintaining high reliability. Other camera types are not configured in this guide.

Setup & Database Login

Step 1: Ensure that the software and MySQL are installed on your system. On the first run, if the database does not exist, the system will automatically create the necessary database and tables.

Step 2: Enter your MySQL login credentials:

  • Host
  • Username
  • Password

After a successful login, if the Users table is empty, the system will automatically create default accounts:

  • SuperUser: superuser / superpass
  • Admin: admin / adminpass
  • Moderator: moderator / modpass
  • User: user / userpass

Control Panel

Once you have successfully logged in to the database, you will be redirected to the control panel. The interface features:

  • Smooth animations and modern rounded inputs/buttons.
  • An option to switch between Dark and Light modes.
  • Logout and exit functionalities.

Depending on your user role (SuperUser, Admin, Moderator, User), the control panel will display different management functions.

Role-Based Features

SuperUser

Full system management: add, edit, delete user accounts, manage students, export data, and adjust system settings.

Admin

Manage student records: add, edit, delete student information, update attendance status, and set cutoff times.

Moderator

Monitor the attendance process and export class-specific reports.

User

View attendance details for assigned students.

Detailed Usage Instructions

Step 1: Setup and Database Login

- Launch the software. If the database is not yet created, the system will automatically set it up.
- Enter your MySQL login credentials (Host, Username, Password) and click "Login".

Step 2: Access the Control Panel

- Once the database configuration is complete, you will be directed to the control panel.
- Use the features according to your assigned role.

Step 3: Manage Students and Attendance

- As an Admin, add students by clicking "Add Student" and selecting the student's image from your computer.
- Edit or delete student records using the corresponding buttons on the list.
- When taking attendance, the system automatically updates the status (on time, late, etc.) based on the cutoff time.

Step 4: Export Data and Set Cutoff Time

- Managers can export the student list as a CSV file for record keeping or reporting.
- Use the "Set Cutoff Time" feature to define a deadline for attendance. Students checking in after the cutoff will be flagged as "Late".
